# CTF RT1 2023-2024

### Contexte

> Dans le cadre d'une SAE option cybersécurité, les 2emes années de RT réalisent un CTF dédié aux 1ère années. Ce dépôt contient les deux épreuves de programmation basées sur des communications TCP simples.

### Les épreuves

> Chaque épreuve à son propre script de serveur d'écoute TCP, respectivement sur les ports 51000 et 51100. 

- Epreuve 1
>- Le participant a à sa disposition un script python à trous. L'objectif est de compléter le code, qui consiste à scraper les données reçues du serveur, réaliser le calcul demandé, puis de l'exécuter pour interroger le serveur et enfin obtenir le flag.
